
## **获取ua对应的手机品牌**

1、当前市面上最全

2、欢迎大家提供ua线索
### 安装

##### Vue
`
npm install pingansec-brandua
`

##### 其他

`dist/brandua.min.js  你懂的
`


### 用法

```javascript
//非vue 省略这行
import BrandUa from "pingansec-brandua";

BrandUa.getBrand('Android HUAWEI');
```
